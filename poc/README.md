# POC — רכיב המפה (MapLibre + OSM ישראל)

הוכחת היתכנות לרכיב המפה המרכזי של היישום. קובץ HTML עצמאי אחד, ללא תלויות בנייה.

## הרצה
```bash
cd field-nav-app/poc
npx http-server -p 5050 -c-1
# פתיחה: http://localhost:5050
```

## מה מודגם
| יכולת | דרישה | מימוש ב-POC |
|---|---|---|
| רכיב מפה אינטראקטיבי | R1 | MapLibre GL, ממוקד ישראל, זום/הזזה/מצפן |
| מפות רקע מתחלפות | R8 | טופוגרפי/שבילים (Israel Hiking Map), צילום אוויר (Esri), מפת רחוב (OSM) |
| עצירות לאורך מסלול | R3 | מסלול לדוגמה + סמני POI (מעיין, עתיקות, תצפית, קפה) עם כרטיסיות |
| אווטאר משתמש | R11 | סמן ג'יפ צבעוני על המפה |
| מיקום המשתמש | R1 | כפתור איתור (Geolocation API) |
| קרדיטים | עיקרון | פאנל מקורות שמתעדכן לפי השכבה הפעילה |
| עיצוב מזמין + מצב לילה | עיקרון | ממשק צף RTL, יום/לילה |

## מקורות (כולם חינמיים)
- **Israel Hiking Map** — אריחי שבילים טופוגרפיים (CC BY-NC-SA). לגרסת ייצור מסחרית: לרנדר אריחים עצמאית מ-OSM.
- **OpenStreetMap** — נתוני בסיס (ODbL).
- **Esri World Imagery** — צילום אוויר.

## דף נוסף — תכנון מסלול חי (R2 + R3)
`route-planner.html` — הדגמה **חיה עם נתונים אמיתיים** של הייחוד המרכזי:
- לחיצה על נקודת התחלה + יעד → **ניתוב שטח אמיתי** דרך BRouter (פרופיל הליכה).
- בניית **מסדרון 1 ק"מ** סביב המסלול (`turf.buffer`).
- שליפת **POI אמיתיים** מ-Overpass (מעיינות, עתיקות, מערות, תצפיות, מי שתייה, קפה, פיקניק).
- **סינון ודירוג** לפי מרחק מהמסלול (`turf.pointToLineDistance`) — זהו ה-corridor search מהאפיון.
- רשימת עצירות לחיצה + סמנים על המפה + כרטיסיות.

אומת: מסלול לדוגמה בהרי ירושלים החזיר 8.6 ק"מ ו-18 עצירות אמיתיות (מעיין מרים, עין צובה, בית גתות ועוד).
פתיחה: `http://localhost:5050/route-planner.html`
